SmartCart - Full-Stack E-Commerce Platform

A modern, full-stack e-commerce application built with "React.js" and "Django REST Framework".

# Live Project Link
You can access the fully functional application here:
[SmartCart Live Website](https://smart-cart-full-stack.vercel.app/)

# Note: The frontend is seamlessly connected to a Django REST API hosted on Render.

 
# Features
- Product Management: Dynamic product listing fetched from Django REST API.
- Shopping Cart: Add to cart, remove items, and real-time total price calculation.
- Secure Payments: Integrated with Razorpay for a smooth checkout experience.
- Responsive UI: Fully mobile-responsive design using modern CSS.
- Robust Backend: Built with Django, supporting CRUD operations and API endpoints.



# Tech Stack
| Frontend | Backend | Database | Deployment |
| React.js | Django (DRF) | SQLite | Vercel & Render |



# Project Structure
- smartcart-frontend: React components, state management, and API calls.
- smartcart_backend: Django settings, models, and REST API views.
- static: Static files for CSS and images.

# Installation & Setup

# Backend Setup
1. Navigate to the backend folder: `cd smartcart_backend`
2. Install dependencies: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`

# Frontend Setup
1. Navigate to the frontend folder: `cd smartcart-frontend`
2. Install packages: `npm install`
3. Start React app: `npm start`
